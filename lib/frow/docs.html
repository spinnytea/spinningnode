Frow Docs

Capture some concepts before/after they make it to code. Useful for devs for new features, useful for players to review how things work.

Core

Regardless of the game we make, these are the core mechanics. The specific mix of these ideas is what creates the game feel and strategy.

Resources

Physical
Rendered, has form and location. Needs to be moved into place to be used.
Intangible
just a counter. can be used from anywhere at any time. they have storage houses like batteries or grain silos.

Actors

These are the dudes that move around. People, robots, etc. If something needs to be in a place, they get it there. So the move resources. They push buttons. They operate machines.

Transmuters

These turn resources into other resources. Like turning Ore into Metal, or Plastic into Money.

Tech Tree

Every tech has prereqs of resources and possibly other tech. Prereqs ought not change between scenarios, but scenarios can define which start unlocked.
